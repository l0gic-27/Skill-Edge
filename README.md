# Skill-Edge
Skill Edge is an AI-powered Resume Analyzer that evaluates resumes, calculates job match scores, identifies skill gaps, and generates personalized interview questions using NLP and LLM technologies. Built with React, Vite, PDF.js, and Groq LLaMA models for intelligent career guidance.

# 🚀 Skill Edge - AI Resume Analyzer

Skill Edge is an AI-powered Resume Analysis platform that helps job seekers improve their resumes, understand their career readiness, and prepare for interviews.

The system analyzes uploaded PDF resumes, extracts skills, evaluates resume quality, identifies skill gaps, calculates job match scores, and generates personalized interview questions using AI.

## 🌐 Live Demo

https://skill-edge-zeta.vercel.app/

## ✨ Features

### 📄 Resume Analysis
- Upload PDF resumes
- Extract text using PDF.js
- Analyze resume structure and content

### 🎯 Resume Scoring
- Score resumes out of 100
- Evaluate:
  - Projects
  - Experience
  - Skills
  - Education
  - Certifications
  - Achievements
  - Internships
  - Interests
  - Hobbies
  - Summary

### 💼 Job Match Analysis
- Detect technical skills automatically
- Compare candidate skills with industry role profiles
- Generate Job Match Percentage
- Recommend missing skills

### 🧠 Skill Detection
Detects 100+ technical skills including:

- Python
- Java
- JavaScript
- React
- Node.js
- MongoDB
- MySQL
- AWS
- Docker
- Kubernetes
- TensorFlow
- PyTorch
- Scikit-Learn
- And many more

### 🤖 AI Interview Questions
- Personalized interview questions
- Domain-specific questions
- Experience-level based questions
- Powered by Groq LLaMA models

### 📊 Interactive Dashboard
- Resume Score Visualization
- Skill Analysis Charts
- Job Match Insights
